# OMD Editor

> **智慧生而共通。** 单文件 HTML 编辑器。零依赖，打开即用。

```bash
npm install && node server.js
# → http://localhost:3456
```

## 四模式

| 模式 | 说明 |
|------|------|
| 💡 默认 | 灵感流块编辑器，Enter 新建块，拖拽排序 |
| 📄 文稿 | A4 Word 风格，所见即所得 |
| 🎞 演讲 | PPT 风格，全屏演示 |
| 📝 源码 | 纯文本 OMD 格式 |

## 功能

📥 Word 导入 · 📤 导出 docx/xlsx/pptx/pdf · 🌓 双主题 · 🔍 Ctrl+F · 🖨 Ctrl+P · ⌨ ? 快捷键 · 💾 自动保存 · 🖱 Ctrl+滚轮缩放 · 📁 Vault 文件管理

## 格式

OMD = YAML 元数据 + GFM Markdown + `<!--OMD-->` 渲染配置。MIT 开源。
