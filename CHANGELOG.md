# OMD Editor v0.9.0

> **智慧生而共通。** 单文件 HTML 编辑器，零依赖，打开即用。

## 🍎 乔布斯灵魂

基于 Steve Jobs「简单比复杂更难」哲学，完成六项灵魂改造。

### 砍掉的
| 砍掉 | 替代方案 |
|------|---------|
| 🌓 主题切换按钮 | 跟随系统 `prefers-color-scheme` |

### 留下的（每一样都有理由）
- **3s 色温过渡**：body、工具栏、侧边栏跟随系统主题平滑渐变
- **块弹跳动画**：cubic-bezier(.34,1.56,.64,1) + 微微旋转入场
- **块纸张质感**：五种底色（idea/todo/done/quote/pin）+ rotate(.1deg) + 多层阴影
- **文稿纸色温**：#fafaf6 暖白 + 纤维纹理 SVG 叠加 + 四层阴影
- **脉冲按钮**：首个新建按钮呼吸光晕，引导首次操作
- **块流渐变淡出**：底部暖色渐变，暗示可继续滚动

## 💡 灵感流（默认模式）

- 块编辑器：图标前缀（💡📝✅📌🔖💬）+ Enter 新建同类块
- 底部 56px 圆形大按钮，移动端优先
- 块拖拽排序
- `# ` 自动格式化（Markdown 快捷输入）
- 5 种块底色（idea/todo/done/quote/pin）
- 选中文字 1.5s 后浮出格式条

## 📄 文稿模式

- A4 纸浮桌面，仿 Word 所见即所得
- 仿宋字体，符合公文排版习惯
- 纸阴影 4 层 + 纸纤维纹理
- Ctrl+滚轮缩放
- 双页并排

## 🎞 演讲模式

- PPT 风格，全屏播放（F5）
- 4 种版式模板（标题页/内容页/两栏/结束页）
- 幻灯片排序（Alt+↑↓）+ 切换动画
- 3 种配色主题

## 🔄 格式互转

- **导出**：docx / xlsx / pptx / pdf（4 端点全部 200 OK）
- **导入**：Word (.docx) 拖入即导入（mammoth.js）

## 🛠 工具

- Ctrl+F 查找栏 · ? 快捷键面板 · 🖨 打印 CSS
- Ctrl+\ 侧边栏（大纲/Vault 文件管理）
- 表格右键菜单 · 图片粘贴 · 自动保存

## 🔌 MCP 接口

- 标准 MCP 协议，外部 Agent 可读写 Vault
